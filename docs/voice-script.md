## Direction notes for actors

- **Tone:** neutral, confident, unhurried â€” think "professional poker tournament dealer," not "sports announcer." No dramatic emphasis unless noted.
- **Pace:** steady. Amounts and actions will be stitched together, so each clip must start and end cleanly with no rising/falling intonation that telegraphs a sentence boundary.
- **Recording specs:** 48 kHz / 24-bit WAV, mono, -18 dBFS average, -3 dBFS peak ceiling. Trim to ~30 ms of silence at head and tail. No room tone, no breaths, no lip smacks.
- **File naming:** lowercase, underscores, `.wav`. Exact filenames are listed per line below.
- **Retakes:** give us three takes per line unless marked otherwise. Slate verbally before each take ("Take oneâ€¦").
- **Do not add filler** ("uh," "okay," "and now"). Do not invent variants. If a line feels awkward, flag it but record it as written â€” we'll adjust the script, not the delivery.

### Performance keys

| Symbol | Meaning |
|---|---|
| `[neutral]` | Flat informational tone |
| `[cue]` | Slightly brighter, attention-getting (used for "your turn") |
| `[positive]` | Warm, upward inflection (wins) |
| `[negative]` | Grounded, downward inflection (losses) â€” do not sound sad, just lower |
| `[terminal]` | Falling tone, end-of-statement |
